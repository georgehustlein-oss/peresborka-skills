# Peresborka Skills

Каталог авторских скиллов для AI-команд по методологии «Пересборка».

## Что это

Skill — упакованная практика команды, которую Claude автоматически использует, когда задача под него подходит. Это переход от «личного промпта» к общему рабочему стандарту команды.

В отличие от большинства публичных коллекций промптов, эти скиллы — формализация конкретной авторской методологии («Пересборка»), не сборник универсальных PM-шаблонов.

## Структура каталога

Каталог разбит на три круга по приоритету внедрения:

- **Круг 1 — Discovery.** Базовые скиллы для работы с идеями и сигналами на этапе продуктового исследования.
- **Круг 2 — Регулярные задачи команды.** Скиллы для повторяющейся ежедневной работы.
- **Круг 3 — Стратегия и внешний контур.** Опционально, после базовой устойчивости команды.

Полный плейбук внедрения и контекст методологии — на [peresborka.io](https://peresborka.io).

## Установка

В разработке: однострочник для установки всего каталога через bash.

На данный момент — ручная установка через копирование `SKILL.md` файлов из нужного круга в `~/.claude/skills/` (для Claude Code) или импорт через настройки Claude.ai.

Подробная инструкция по установке для разных вариантов (Claude.ai, Claude Code, Cowork, организация) — на [peresborka.io/skills/install](https://peresborka.io/skills/install).

## Безопасность

В этих скиллах нет исполняемого кода. Только текстовые инструкции для модели в формате `SKILL.md`.

Это значит:
- Нет сетевых запросов
- Нет доступа к файловой системе
- Нет вызовов внешних API
- Нечему «выстрелить» как вредоносному коду

Каждый файл можно прочитать целиком до установки — никаких скрытых частей, никаких бинарников.
